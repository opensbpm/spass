Object Property: requiresPerformedMessageExchange

Annotations (1)

  • rdfs:comment "This DataProperty links a send or receive transition condition to a specific message exchange.

    For send transition it implies that the according message has been send to the in-box of the corresponding receiver subject.

    For receive transitions it implies that the according message from the corresponding sender subject is in the in-box and could be extracted.

    When a MessageExchangeCondition is not fully specified (missing sender, receiver, or message) it cannot reference a specific message exchange.

    In that case the individual reference must contain the according information:

    requiresSendingOfMessage (SendTransition)
    requiresMessageSentTo (SendTransition)
    requiresReceptionOfMessage (ReceiveTransition)
    requiresMessageSentFrom (ReceiveTransition)"     @en
